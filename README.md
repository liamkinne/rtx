# UMI RTX Robot Arm Refit

I have a UMI RTX with a faulty control board due to it being damanged when the University of Tasmania engineering building flooded in 2018. In this repo is firmware and other tools for using my replacement control board.

The new control board is designed in KiCad, based on the STM32G474, with firmware written in Rust. The goal is to enable the arm to be controlled by host computer via GCODE commands to execute programs or perform real-time joint control.
